# Sistema La Esperanza

Repositorio del proyecto de análisis y diseño del sistema de gestión agrícola para la Asociación La Esperanza.

## Alcance

Este repositorio documenta los modelos principales del sistema, enfocados en:

- Interacción de actores con funcionalidades clave del sistema.
- Estructura de entidades de datos y sus relaciones.

## Modelos

- Modelo de Casos de Uso: [`docs/modelo-casos-de-uso.md`](docs/modelo-casos-de-uso.md)
- Modelo Entidad-Relación (ER): [`docs/modelo-er.md`](docs/modelo-er.md)

## Notas

- Los diagramas están en formato Mermaid para facilitar su visualización y mantenimiento.
- Se recomienda visualizar este repositorio en plataformas compatibles con Mermaid (por ejemplo, GitHub).

## Autores
